# Teacher Assistant App | مساعد المعلم المساعد

A comprehensive mobile-first application designed for teachers to manage students, attendance, quizzes, results, and financial records with ease and intelligence.

تطبيق متكامل مصمم خصيصاً للمعلمين لإدارة الطلاب، الحضور، الاختبارات، النتائج، والسجلات المالية بكل سهولة وذكاء.

---

## 🚀 Key Features | المميزات الأساسية

### 📊 Smart Dashboard | لوحة التحكم الذكية
- **Data Analytics**: Real-time insights into total students, groups, activity, and financial summaries.
- **Smart Suggestions**: Proactive prompts based on attendance trends, unpaid fees, or upcoming sessions.
- **تحليل البيانات**: إحصائيات مباشرة حول عدد الطلاب، المجموعات، النشاط، والملخص المالي.
- **الاقتراحات الذكية**: تنبيهات استباقية بناءً على الغياب، الدفعات المتأخرة، أو الحصص القادمة.

### 👥 Group & Student Management | إدارة المجموعات والطلاب
- **Dynamic Groups**: Manage weekly schedules, locations, and pricing per group.
- **Comprehensive Profiles**: Track student personal info, contact details, attendance history, and payment status.
- **المجموعات الديناميكية**: إدارة المواعيد الأسبوعية، الأماكن، والأسعار لكل مجموعة.
- **ملفات الطلاب الشاملة**: تتبع بيانات الطلاب، أرقام التواصل، سجل الحضور، وحالة الدفع.

### 📝 Academics & Quizzes | الاختبارات والتقييم الأكاديمي
- **Quiz Builder**: Easily create quizzes, record scores, and track student performance over time.
- **Honor Board**: Automatic identification of top-performing students to encourage excellence.
- **بناء الاختبارات**: سهولة تسجيل الدرجات ومتابعة تطور مستوى الطلاب.
- **لوحة الشرف**: تحديد تلقائي للمتفوقين لتشجيعهم.

### 💰 Financial Management | الإدارة المالية
- **Payments Tracking**: Log fees, manage discounts, and see outstanding balances per month.
- **Expenses Recorder**: Track educational expenses and business costs.
- **سجل الدفوعات**: تسجيل المصاريف، إدارة الخصومات، ومعرفة المتأخرات الشهرية.
- **سجل المصروفات**: تتبع مصاريف التدريس وتكاليف العمل.

### 📱 Smart Communication | التواصل الذكي
- **WhatsApp Reminders**: One-click messaging for attendance alerts and payment reminders.
- **Student Portal**: Secure landing page for students/parents to view their grades and attendance records.
- **رسائل واتساب**: ضغطة واحدة لإرسال تنبيهات الحضور وتذكيرات الدفع.
- **بوابة الطالب**: صفحة مؤمنة للطلاب وأولياء الأمور لمتابعة النتائج والحضور.

---

## 🛠 Tech Stack | التقنيات المستخدمة

- **Framework**: Flutter (Material 3)
- **State Management**: Riverpod
- **Database**: Firebase Firestore (Offline-first architecture)
- **Authentication**: Firebase Auth & Local Auth (PIN/Biometrics)
- **Architecture**: Clean, feature-based modular folder structure.

## 🛠 Development Setup | إعداد التطوير

### Prerequisites
- Flutter SDK (v3.32+)
- Dart SDK
- Firebase account & CLI

### Installation
1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Setup Firebase:
   - Run `flutterfire configure` to link your Firebase project.
4. Run the app:
   ```bash
   flutter run
   ```
